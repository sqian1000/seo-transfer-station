# /go/ 点击追踪方案

`/go/` 只用于外部资源中转与点击意图统计，不应作为可收录内容页。SEO 主内容应放在首页和专题页。

## P0 规则

1. `/go/` 页面必须 `noindex,nofollow`。
2. `/go/` 不进入 sitemap。
3. 外链使用 `rel="nofollow noopener noreferrer"`。
4. 不做 0 秒自动跳转，避免形成 doorway 页面。
