# AI API 中转站模型与价格矩阵

本页是价格与模型信息的安全占位模板。真实价格、倍率、可用模型、上下文长度和状态页 URL 必须由业务侧确认后再发布。

## 建议字段

| 字段 | 说明 |
|---|---|
| 模型名称 | 控制台展示名和真实 model id |
| 上游类型 | Claude / OpenAI compatible / Gemini / DeepSeek 等 |
| 价格倍率 | 待接入真实控制台 |
| 额度与余额 | 以控制台实时展示为准 |
| 是否支持流式输出 | 待确认 |
| 限流规则 | RPM、TPM、并发和重试建议 |

## 合规提醒

不要承诺“绝对稳定”“长期免费”“封禁风险归零”“不限额度”。更稳妥的表达是：可用性受上游、线路、套餐和维护窗口影响，生产环境应准备降级策略。
