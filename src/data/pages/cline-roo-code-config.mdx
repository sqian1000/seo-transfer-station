# Cline / Roo Code 配置入口

Cline、Roo Code 等客户端通常通过 provider、Base URL、API Key 和 model 字段接入 OpenAI 兼容 API。

## 建议配置

- Provider：选择 OpenAI compatible 或自定义兼容接口。
- Base URL：`https://YOUR_API_DOMAIN/v1`。
- API Key：控制台生成的项目 Key。
- Model：控制台展示的 model id。

## 排错

若出现 `model not found` 或 `429 rate limit`，先回到首页排错矩阵，再检查套餐权限和限流规则。
