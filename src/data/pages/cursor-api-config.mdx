# Cursor API 配置入口

Cursor 是否可用自定义 API，取决于当前客户端版本和 provider 设置。配置前先确认是否支持 OpenAI compatible Base URL。

## 检查项

- `base_url` 使用 `https://YOUR_API_DOMAIN/v1`。
- API Key 来自已确认控制台。
- 模型名使用控制台展示值。
- 出现 401、404、429 时回到首页排错矩阵。
