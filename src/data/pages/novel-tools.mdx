# 后续实验室

当前主线不是小说站，也不是小说阅读站，而是 **SEO 中转站推广与点击验证系统**。

这里仅保留后续可接入方向：当某类入口在 `/go/` 点击数据中表现足够明确，再考虑把它升级为自研工具、表单、API 或 SaaS 功能。

## 可能扩展方向

1. **长文本创作工作流**：大纲、人设、章节检查、长文记忆。
2. **小说 / 短剧实验工具**：小说大纲、前三章诊断、短剧分集拆解。
3. **多 Agent 内容生产**：选题、素材、草稿、质检、发布辅助。
4. **SEO 自动化工具**：关键词聚类、内链规划、标题描述生成。

## 当前规则

- 首页和主导航优先服务 SEO 中转站。
- 小说能力只作为未来实验，不抢首屏、不抢主 CTA。
- 先看搜索流量和中转点击，再决定是否开发重功能。
