# 线路状态面板说明

状态页用于区分“配置错误”和“服务波动”。真实状态页 URL、可用性指标和维护公告系统尚未接入前，本页仅作为结构占位。

## 建议展示

- API 网关状态。
- 上游模型状态。
- 备用线路状态。
- 近期维护公告。
- 错误码趋势和延迟区间。

## 合规提醒

不要伪造稳定率、成功率或 SLA。所有可用性信息都应来自真实监控或人工确认。
