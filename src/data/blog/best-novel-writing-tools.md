# 最好用的小说写作工具推荐

小说写作工具不应该只负责“生成文字”，更重要的是帮助作者理清大纲、人设、冲突和章节节奏。对于个人作者来说，最值得优先尝试的是大纲工具、人物设定卡、章节标题生成器和前三章诊断清单。

## 推荐工具类型

1. 大纲生成工具：适合新书立项和卡文时重建主线。
2. 人物设定工具：适合长篇连载，避免人设前后矛盾。
3. 章节标题工具：帮助每章建立明确钩子。
4. 世界观清单：适合玄幻、科幻、无限流和民俗题材。
